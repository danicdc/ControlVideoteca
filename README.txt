El proyecto es una aplicación que es una videoteca personal que me muestra mis peliculas con una ficha de cada pelicula.
# Profe:
Indicar que será realizado en C# y la interfaz con ayuda de XAML, además la posibilidad de extensiones de plugins en xml.
